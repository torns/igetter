# Downloader

Downloader 对于使用者来说只需实例化再连接至 Engine 即可。

# 方法

## Downloader()

构造一个 Downloader 实例。

- **参数**：无
- **返回类型**：Downlaoder
- **用法**：

```ts
let downloader = new Downloader()

```

## attachEngine(engine)

连接至 Engine 。

- **参数**：Engine
- **返回类型**：无
- **用法**：

```ts
downloader.attachEngine(engine)

```
